# TicTacToe

Traditional Tic Tac Toe game implented in Ruby

## Built With
​
Ruby 2.6.5
​
## Getting Started

​To get a local copy up and running follow these simple example steps.​ 
Git clone https://github.com/epinczinger/TicTacToe

gem install colorize

## Commands

ruby main.rb
